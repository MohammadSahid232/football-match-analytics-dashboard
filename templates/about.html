Nepal Football Match Analytics Dashboard

A professional web application designed for a University Software Engineering & Data Analytics project.

This dashboard integrates Pandas for data preprocessing, Scikit-learn for machine learning match predictions, Matplotlib/Seaborn for data visualizations, and Flask for routing. It utilizes synthetic dataset representing the top clubs in Nepal.
